# Travel Guru
## Overview

- This is a simple website for travelling agency.
- Authenticated users can select a location.
- Different hotels are displayed in Google Map according to the chosen destination.
- Users can book any hotel.

## Technology

- React
- react-router-dom
- react-hook-form
- Firebase auth
- Firebase hosting
- CSS
- BootStrap
- Leaflet
- font-awesome
